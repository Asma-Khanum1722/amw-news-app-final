## ğŸ“¦ Installation
Get this running on your local machine in less than 2 minutes.

### 1. Clone the repo
```bash
git clone https://github.com/Asma-Khanum1722/amw-news-app-final.git
cd amw-news-app
```

### 2. Install dependencies
```bash
npm install
```

### 3. Configure Environment
Create a `.env` file in the root directory:

```env
# GNews API Key
VITE_GNEWS_API_KEY=your_gnews_api_key

# Firebase Configuration (If applicable)
VITE_FIREBASE_API_KEY=your_firebase_key
# ... other firebase config
```
*(You can get a free API key from [GNews.io](https://gnews.io/))*

### 4. Start the app
```bash
npm run dev
```
